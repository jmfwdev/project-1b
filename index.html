CSS Transformations

CSS property that allows you to appply a 2D or 3D transformation to an element, lets you rotate, scale, skew and translate an element. You can create animation by combining transforms with transitions

Rotate

rotates element clockwise or counter clockwise at a given degree

Skew

shorthand for skewX()skewY() allows you to tilt an element one way or another

Scale

can increase or decrease the size of elements using this function

Translate

moves element from current position along x and y axis

By modifying the coordinate space, CSS transforms change the shape and position of the affected content without disrupting the normal document flow
