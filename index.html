CSS Animations

A css animation can be a unique way to add uniqueness to your projects and manipulate the users attention to where you want it. A Css animation is a way to change the appearance or position of an element over a period of time. This, is done using keyframes which are the to and from states of this change. The animation is then created when moving between these to stated keyframes.

Step One

First you must define the keyframes. Basically describing the keyframes step by step.

Step Two

You must apply the animation to the choosen element. Use the animation property to define keyframes to an HTML element.
